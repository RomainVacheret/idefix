# IDEfix

## Prerequisites
*Note: Developed for Unix-based operating systems*

- Git
- JDK 8
- Maven
- Python 3.9

## Installation
```bash
git clone https://github.com/RomainVacheret/idefix
cd idefix
./install_dependencies.sh
python3 -m pip install -r requirements.txt

# Start the project with
python3 idefix.py
```
